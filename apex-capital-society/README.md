# Apex Capital Society

A professional React landing page for Apex Capital Society (ACS), a student-led investing and financial literacy organization founded in Tampa.

## Run locally

This implementation is intentionally dependency-light because the current workspace does not expose `npm` on PATH. It uses editable React JSX, Tailwind CSS CDN, Framer Motion ESM CDN, and subtle interaction patterns.

```powershell
node .\serve.mjs
```

Then open `http://localhost:4173`.

## Interaction notes

The site includes a restrained cursor glow, flowing navigation, and a visual learning-topics section focused on practical finance concepts like ETFs, funds, retirement accounts, risk, and long-term investing.
